To make the OCR AI work for images, you MUST place the Tesseract trained data file here.

1. Download this file: https://github.com/tesseract-ocr/tessdata/raw/main/fra.traineddata
2. It's about 15 MB. Save it right next to this file, named exactly "fra.traineddata".
3. That's it! 

(If you want it to also read English, download `eng.traineddata` from the same GitHub repo).
